The Hello World Tradition

As any coder would tell you, the very first project you will always do is called the "Hello World" project.

Brian Kernighan actually wrote the first "hello, world" program as part of the documentation for the BCPL programming language developed by Martin Richards. BCPL was used while C was being developed at Bell Labs a few years before the publication of Kernighan and Ritchie's C book in 1972.

The "Hello World" project varies in its implementation. For Arduino hobbyists, the "Hello World" project involves getting an LED to light up. For those starting with HTML, you get a webpage that says:

Hello World

The Hello World project will always provide coders with a foundation for beginning in their chosen language.

Making Pretty

After an initial HTML class, you will very quickly begin learning about CSS, which is a language that deals with the visual elements of a website. At first you will utilize visual elements utilizing a style element. Eventually you will utilize reference elements that will reference a seperate file that contains your CSS: link rel="stylesheet" type="text/css" href="documentation.css"

Background Images

Backgound images..are a dick to do. The code is logically very simple, but I learned quickly that simply putting the code into CSS will not, a background image make. The code is very simple:

body { background-image: url("texture.jpg");

background-repeat: no-repeat;

background-position: fixed;

background-size: cover;}

Unfortunately the ORDER in which you put code is a vital element, and it took me several tries to get it right, and I am not certain I truly have it sorted.
